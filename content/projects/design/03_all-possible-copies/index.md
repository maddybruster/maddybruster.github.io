All Possible Copies is a bootleg version of Jon Sueda's exhibition catalogue [All Possible Futures](http://allpossiblefutures.net). All Possible Futures explores speculative work created by contemporary graphic designers – formally expressed by the creation of eight parallel exhibition catalogues: All Possible options, All Possible Evils, All Possible Errors... we generated a tenth speculative catalogue. Held in the collection of the Pratt Institute Library. Designed and produced with [Carter Gekiere](https://www.cartergekiere.info), 2018.
